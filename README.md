# 🏦 Nova Bank - Banking Management System

A modern Banking Management System developed using **Python, Flask, HTML, CSS, and Bootstrap**. The project provides basic banking operations through a clean web interface.

---

## 🚀 Features

- ✅ Create New Account
- 💰 Deposit Money
- 💸 Withdraw Money
- 🔄 Transfer Money
- 🔍 Search Account
- 📋 Display All Accounts
- 📝 Transaction History
- 💾 Data stored using JSON

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Bootstrap 5
- JSON

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Nova-Bank.git
```

Move into the project

```bash
cd Nova-Bank
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
